# Stack
1. Ruby 3.1.1
2. Ruby on Rails 7.1.3
3. Postgresql
4. Action Cable
5. Stimulus
6. bootstrap

## Installation

### Development setup

1. bundle install
2. copy the .example.env to .env
3. rails db:create
4. rails db:migrate
5. bin/dev

### Flow
1. sign up / login
2. create the room
3. waiting another player
4. play the game
